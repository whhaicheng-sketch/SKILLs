# SKILLs

个人 Codex Skills 仓库。

## Skills

### mysql-bug-analysis

用于调查、复现和分析 MySQL BUG 的 Codex Skill，支持：

- MySQL 官方 BUG、官方文档、Release Notes 和官方源码证据调查
- 受影响版本、修复版本和修复 commit 定位
- Debug Build、隔离实例、MTR、GDB、Core Dump 和源码插桩
- 无法自然复现时的故障注入、补丁反推与静态源码分析
- L1～L5 结论可信度评级
- 自动生成 BUG 分析报告和复现调试报告
